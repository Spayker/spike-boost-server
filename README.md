# SpikeBoost

## Description
SpikeBoost helps to solve the issue with annoying searching of best soundtrack in list.
Especially when you take part in some physical activities and you have a choice to listen music simultaneously.

The project has proof of concept state now. for this purpose it has two parts:
1) server side
2) client side

Basically client side will be deployed on mobile platforms such as Android, IoS.

### Stack of technologies

For server side:
1) Java 8
2) Spring Boot 
3) Rest
4) Spring Security
5) Junit 5
6) Mockito

For client side:
1) React-Native
2) Java 8
3) Android Min SDK: 21
4) TensorFlow Lite
5) Test Espresso
6) Test Instrumentation
